REST API Reference#

Note

See the Glossary for the meaning of the acronyms used in this guide.

This page documents the endpoints and available HTTP methods for the Dioptra REST API. In addition to using this page, it is highly recommended that Dioptra users also use the Swagger documentation that the REST API service automatically generates at runtime, which presents all of this page’s information in an interactive format. To access the Swagger documentation, just navigate to the web URL for the Dioptra REST API service (omit the /api/v1 part at the end of the web address).
